# Lógica de Programação - Lista 4 (Operadores Lógicos) 👥

Repositório criado para armazenar as resoluções da **Lista 4** da disciplina de Lógica de Programação e Algoritmos. As atividades focam no uso de operadores lógicos complexos para tomada de decisão.

## 📝 Exercícios Desenvolvidos

A lista aborda os seguintes cenários de decisão:

1.  **Aprovação Acadêmica**: Checa nota $\ge 7.0$ **E** frequência $\ge 75\%$.
2.  **Entrada Gratuita**: Verifica se a idade é $< 5$ **OU** $\ge 65$.
3.  **Controle de Estoque**: Validação de disponibilidade de produto para venda.
4.  **Análise de Currículo**: Critério de idade mínima **E** conhecimento em Python **OU** JavaScript.
5.  **Aprovação de Empréstimo**: Verifica renda ou fiador, com a condição obrigatória de **NÃO** ter nome sujo.
6.  **Sistema de Super Desconto (Bônus)**: Lógica complexa combinando status VIP, valor de compra, cupons e histórico de pagamentos.

## 📁 Estrutura do Repositório
* Arquivos `.alg`: Códigos fonte em pseudocódigo.
* Arquivos `.fprg`: Fluxogramas editáveis no Flowgorithm.
